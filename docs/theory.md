# AOD Theory Reference

## Categorical State

- Layers as types
- Objects as points
- Morphisms as paths with weight and probability

## AOD Dynamics

- Growth: G = A e^{-k S}
- Efficiency η = G/A
- Governed recurrence: η_{t+1} = η_t e^{-k/η_t} + γ(η* - η_t)

## Spectral Gap

The dominant eigenvalue is ~0.93, giving exponential convergence with gap ~0.07.

## Univalence

DCC (same ordered log → equal state) is a constructive form of univalence.
